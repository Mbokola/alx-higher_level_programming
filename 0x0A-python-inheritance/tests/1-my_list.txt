=====================
The 1-my_list module
=====================

The module contains only one class called MyList that inherits from list class
Test that examples work fine

First, import the module
>>> MyList = __import__('1-my_list').MyList
>>> #List that is already sorted
>>> MyList([1, 2, 3, 4, 5]).print_sorted()
[1, 2, 3, 4, 5]
>>> #List sorted in descending order
>>> MyList([5, 4, 3, 2, 1]).print_sorted()
[1, 2, 3, 4, 5]
>>> #List with a zero
>>> MyList([5, 4, 3, 2, 1, 0]).print_sorted()
[0, 1, 2, 3, 4, 5]
>>> #List with a string
>>> MyList([5, 4, 3, 2, 1, 0, 'String']).print_sorted()
Traceback (most recent call last):
	  ...
TypeError: '<' not supported between instances of 'str' and 'int'
>>> #List with largets number in middle
>>> MyList([2, 3, 5, 1, 4]).print_sorted()
[1, 2, 3, 4, 5]
>>> #List with a float
>>> MyList([3.3, 2, 0, 4.6, 4.67]).print_sorted()
[0, 2, 3.3, 4.6, 4.67]
